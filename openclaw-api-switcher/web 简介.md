# OpenClaw API Switcher

**为 OpenClaw 打造的专业级 API 模型切换与诊断工具**

---

## 🎯 一句话介绍

OpenClaw API Switcher 是一款专为 OpenClaw 用户设计的桌面应用，让你轻松管理和切换 20+ 个 AI 模型供应商，一键完成模型配置，告别手动修改配置文件的烦恼。

---

## ✨ 核心亮点

### 🚀 极速切换
- **一键切换模型** - 点击即可切换 Provider 和 Model，实时生效
- **自动同步配置** - 保存后自动同步到 OpenClaw，无需手动操作
- **智能备份恢复** - 配置变更前自动备份，随时一键恢复

### 🎨 精美界面
- **琉璃风格设计** - 渐变背景、光晕效果、毛玻璃卡片，视觉享受
- **直观操作** - 标签式导航，清晰的功能分区，零学习成本
- **实时反馈** - 状态栏实时显示 Gateway 服务状态

### 🔧 强大诊断
- **系统健康检查** - 6 项全面检查，快速定位问题
- **一键修复** - Gateway 服务异常？一键自动修复
- **请求追踪** - 实时监控 API 请求，排查连接问题

### 🌍 广泛兼容
- **20+ 供应商** - 支持国内外主流 AI 平台
- **三大分类** - 国内 API / 国际 API / 本地部署
- **自定义扩展** - 支持添加自定义 API 供应商

## 🚀 快速开始

### 1. 下载安装

```bash
# 克隆仓库
git clone https://github.com/yourusername/openclaw-api-switcher.git

# 进入目录
cd openclaw-api-switcher
```

### 2. 启动程序

- **macOS**: 双击 `启动.command`
- **Windows**: 双击 `启动.bat`
- **Linux**: 运行 `./start.sh`

### 3. 配置 API

1. 点击 **"添加 Provider"**
2. 选择供应商（如 Moonshot）
3. 填写 API Key
4. 选择模型，保存配置

### 4. 开始使用

1. 点击 Provider 卡片选择模型
2. 点击 **"应用配置"**
3. 配置自动同步到 OpenClaw
4. 开始使用 OpenClaw！
